68 // Friend is used for commutative binary operators where one of the operators is a built-in type. This essentially 
69 // puts these operators outside of the class where they belong. If the other operand type is user-defined, the 
70 // operator should be completely outside as there would be ambiguity as to which one owned the friend declaration. 
71 tVector3& operator=(const tVec2& v) { tSet(d&: *this, s: v); return *this; }
